# EDC Consumer Module

This module contains the Consumer interface for the Eclipse Dataspace Connector (EDC) management application.

## Overview

The Consumer interface allows users to discover, request, and use assets shared by providers. It provides a user-friendly interface for browsing available assets and managing data usage agreements.

## Key Features

- Asset discovery
- Contract negotiation
- Data usage tracking
- Language switching (English/Spanish)

## Folder Structure

- `/components`: UI components specific to the Consumer interface
- `/lib`: API and utility functions for the Consumer interface

## Current Status

This module is currently under development. The basic structure is in place, but the functionality is not yet implemented.

## Future Development

Future versions will include:
- Asset catalog browsing
- Contract negotiation workflow
- Data transfer initiation
- Usage control monitoring
