# Consumer Components

This folder will contain React components specific to the EDC Consumer interface.

## Planned Components

### AssetCatalogBrowser

A component for browsing available assets from providers.

### ContractNegotiationForm

A component for initiating and managing contract negotiations with providers.

### DataTransferInitiator

A component for initiating data transfers after contracts are established.

### UsageControlMonitor

A component for monitoring data usage according to contract terms.

## Current Status

This folder is currently empty as the Consumer interface is under development.
